###  배경

셀프 네일의 수요가 많아짐에 따라 내 손톱에 어울리는 지 미리 체험하는 수요가 생기게 되었고, 네일아트는 내 손톱에 어울리는지 해보고 결정할 수 없다. 때문에 자신의 손 사진을 찍어서 디자인을 입힌 모습을 보기 위해 이 서비스를 개발하게 되었다
